Hot on the heels of last week’s big release, this update is all about polish: the Flight Tracker learns the map, Radio gets a way to hear everything at once, and the whole desktop got quieter, more keyboard-friendly, and easier to pick up where you left off.

Highlights:

  • Airports on the map — the Flight Tracker now plots airport markers (clustered, with a dedicated Airport Details pane and per-layer toggles), and colors the four notable flights’ tracks by phase of flight in both 2D and 3D.
  • Radio “All Traffic” — a single view that aggregates every non-live station, plus much smoother scrubbing.
  • TV remembers your channels — your single-view and MultiView selection now survive a reload.
  • README tags — articles carry colored tag pills, with a File → Settings panel to show or hide tags you care about.
  • A more accessible desktop — pop-up menus are fully keyboard-navigable, tool palettes now behave like real Mac OS 8 windoids, a new Drive Setup control panel manages synced drives, and a long-standing text-input typing bug is fixed.

Apps

Flight Tracker

  • Airport markers — airports now appear as map pins in both 2D and 3D, clustering as you zoom out, with a per-layer Cluster toggle in the Layers window and a dedicated Airport Details pane. Selecting a pin surfaces its details alongside the flight it belongs to.
  • Phase-of-flight coloring — the tracks for AA11, UA175, AA77 and UA93 are now colored by flight phase (taxi, takeoff, climb, cruise, descent, approach and so on), on both the 2D track line and the 3D altitude tube, so you can read each flight’s profile at a glance.
  • 8-bit radar in 3D — the retro “radar” map style now carries into 3D with pixelated plane sprites, and you can right-drag to tilt the camera past its resting pitch for a lower, over-the-shoulder view.
  • A disclaimer now notes where individual flight tracks have been synthesized from Bureau of Transportation Statistics data.

TV

  • TV now remembers your channel selection across reloads — both a single focused channel and a full MultiView grid, including which tiles are muted — by persisting your picks as stable channel slugs rather than transient ids.

Radio

  • All Traffic — a new view that aggregates every non-live station into one continuous feed, so you can monitor the whole dial at once.
  • Smoother scrubbing — a 10× larger look-ahead window keeps items from vanishing during small rewinds and seeks.

RadioScanner

  • The focused player now has a seekable progress slider showing playable duration, with its position announced for screen readers.

README

  • Tags — articles now display colored, contrast-aware tag pills.
  • Tag filter — a File → Settings window lets you show or hide articles per tag, and your choices persist between visits.

Alerts

  • Emergency-alert dialogs now ring the classic Sosumi chime when they appear.

Across the desktop

Several improvements land at the Classicy desktop level, so they show up everywhere at once:

  • Keyboard-navigable pop-up menus — every dropdown now opens and moves by keyboard with native-style type-ahead, renders unclipped on top of its window, and exposes the highlighted option to assistive technology.
  • Tool-palette windows — utility windows (like the Flight Tracker’s Layers and toolbar palettes) now use authentic Mac OS 8 half-height title bars and stay tucked behind the app you’re focused on, instead of stealing focus.
  • Drive Setup — a new control panel that lists mounted drives and manages your synced account drive.
  • Text-input fix — a bug that stopped some fields from accepting more than a single character is resolved.

Behind the scenes: the streamer now caches its per-connection source lookups, the Directus content backend was hardened for scale, Time Machine scrubbing was tightened, and a new pipeline automatically ships Classicy component updates into the site as they’re published.
